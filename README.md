# 亚马逊仓储费计算器使用说明

## 打开方式

直接双击 `index.html`，用浏览器打开即可使用。

这是单文件静态工具，不需要安装软件，也不需要启动服务器。

## 单品计算需要填写

- SKU
- 国家
- 计费季节：淡季或旺季
- 仓储类目：仅在页面提示需要时填写
- 是否危险品
- 商品包装后尺寸：长、宽、高，单位为 cm
- 商品重量，单位为 kg
- 单品数量
- 单品存放/周转天数
- 计费天数，通常填 30
- 仓储利用率周数：没有后台数据时填 0
- 库存库龄

## 说明

- 页面会按国家规则自动判断标准尺寸或大件。
- 费率表可在页面下方查看并导出 JSON；费率与政策绑定，页面内不可编辑或导入覆盖。
- 批量计算可使用页面内的 CSV 模板入口。
- 官方来源链接在“费率表 - 官方链接”中查看。

## 分享建议

把整个文件夹或 `amazon_storage_fee_calculator_share.zip` 发给同事。同事解压后双击 `index.html` 即可。
